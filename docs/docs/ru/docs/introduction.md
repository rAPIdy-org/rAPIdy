# Introduction

Добро пожаловать в документацию по **<span class="base-color">Rapidy</span>**. 

**Rapidy** - это **простой** и **быстрый** веб-фреймворк для создания лаконичных <br/>
**HTTP API-серверов** со встроенными **валидацией**, **сериализацией** и **асинхронностью**. 

!!! tip ""
    Быстрый обзор возможностей представлен в разделе **[Quickstart](quickstart.md)**.


**Rapidy** базируется на двух других фреймворках:

- <span class="note-color">aiohttp</span> - работа с **web**
- <span class="note-color">pydantic</span> - **валидация** и **сериализация** данных

Посколько **Rapidy** зависит от инструментария **aiohttp** и **pydantic**, при поиске информации
обязательно ознакомьтесь с их документацией, а также с документацией **Rapidy**.

!!! tip ""
    Документация **<a href="https://github.com/aio-libs/aiohttp">aiohttp</a>**.

!!! tip ""
    Документация **<a href="https://github.com/pydantic/pydantic">pydantic</a>**.

!!! tip "**Общие шаблоны описаны в разделе [Examples](examples.md)**"

В остальных разделах документации подробно описывается каждый компонент Rapidy.
